The SVC algorithm is a machine learning algorithm that works using "Support Vectors" - Support Vector Classification

The SVC algorithm is a classification algorithm that works by finding the best hyperplane to classify the data

A hyperplane is subspace that is one dimension less than the dimension of the data

For example, if our data was in 3D, the hyperplane would be a regular plane (2D)

If you look at 2D data, the hyperplane would be a line (1D)

The SVC algorithm finds the hyperplane that best classifies the data

The SVC algorithm is a classification algorithm that works by finding the best hyperplane to classify the data

There are several methods (formally called "kernels") to calculate the hyperplane, the most popular by far are Linear, Poly, and rbf (Radial Basis Function)

SVC works by using two Support Vectors (Hence the name), They pick two points which describe the separation of the data the best way, and then they create the hyperplane between the two. with a margin between each of them trying to be as exact as possible in the middle

How one might choose how big or small should the margin be? We have a parameters "C" - Regularization parameter - the "C" parameters defines how regularized should the hyperplane be. For example in 2D defines how "hard" should be the margin between the Support Vectors

A low value of C creates a soft margin as shown below

A high value of C creates a hard margin as shown below

A low value for C - soft margin - is used to get a general boundary and not to give so much attention to the outliers.

The problem that rises with a low C is that some points are considered on purpose False, to get a general classification - Called Under-Fitting.

Under-Fitting is when you are not giving enough attention to details.

A high value for C - hard margin - is used to get a more exact boundary and not to give so much attention to the outliers.

The problem that rises with a high C is that we are missing a lot of general space because of specific outliers, which are effecting our chances of getting a general model. This problem is called Over-Fitting

Over-Fitting is when are model is not general enough and depends too much on our data. Giving too much attention for outliers
